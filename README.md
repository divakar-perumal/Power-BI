# Power-BI
Overview:
This Financial Dashboard is a interactive and visual representation of key financial metrices which help the viewers to understand the trend, profit, units solds and KPIs in a dynamic format.
I have used the Power BI Desktop version which has some limited features, with that i made a good dashboard.

Features:
* Sales, Profit, Unit sold Insights
* We can use slicers and visualize the charts by year wise, country wise, segment wise.  
* Navigation through pages
* Used Hyperlinks

Requirements:
To access, update and alter anything in this dashboard we need a app Power BI Desktop - https://www.microsoft.com/en-us/power-platform/products/power-bi/desktop

How to Use:
1. Download this Financial Dashboard.pbix file.
2. Open the Financial Dashboard.pbix file in Power BI Desktop.
3. Refresh the report to load your data.
